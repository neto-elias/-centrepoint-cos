# Centrepoint COS

Centrepoint Operating System is a simple DOS-style council governance portal for Strata K518.

## Current version
v0.5 static PWA. It runs on GitHub Pages and stores data locally in the browser.

## Files
- `index.html` - main app
- `assets/style.css` - terminal design
- `assets/app.js` - local data, add/edit/delete/search/export/import
- `manifest.json` and `sw.js` - installable/offline PWA support

## Important
Data is stored on each device only. Use Export before replacing files or changing phones.
